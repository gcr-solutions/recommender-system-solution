## API 概览 

此推荐系统的API共有三种：

- retrieve 
  
  `/retrieve/{userId}` - 获取用户的推荐商品列表 
  
- event
  
   `/event/portrait/{userId}` - 更新用户画像

   `/event/recall/{userId}`  - 触发用户召回

- admin
   
   `/event/start_train` - 触发离线训练任务
  
   `/event/batch_update` - 触发离线批量更新任务，为所有用户重新生成推荐项目列表
