AWS Cloud9

AWS Cloud9 is a cloud-based integrated development environment (IDE) that lets you write, run, and debug your code with just a browser. It includes a code editor, debugger, and terminal.

Get started with AWS Cloud9

  1. Go to AWS Cloud9 Console
  2. Click “Create environment” button to create an cloud9 environment

  1. Name it rs-workshop, click Next
  2. Take all default values and click Create environment
