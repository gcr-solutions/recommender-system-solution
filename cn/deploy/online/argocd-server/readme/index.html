创建 Argo CD 服务器

在本节中，您将在 EKS 集群中创建 Argo CD 服务器

  1. 跳转到 /home/ec2-user/environment/recommender-system-solution/scripts 目录
cd /home/ec2-user/environment/recommender-system-solution/scripts
  1. 运行以下命令在 eks 集群中设置 Argo CD 服务器
./setup-argocd-server.sh

这将需要大约 1 分钟的时间进行配置

控制台将输出 argocd 的用户名、密码和服务器端点URL，如下所示：

将端点URL复制到浏览器中访问 Argo CD 服务器网页，如果您是第一次访问此端点，请单击Advanced和Proceed to …

输入用户名和密码，然后点击登录，网页应该如下图所示：

Argo CD 服务器设置成功！!
